Usando Links Externos

A duas maneiras de adicionar links externos. utilizando a tag A, o HTML cria uma Âncora onde coloca a referência do link externo que você quer mandar o usuário.

Você pode Acessar meu repositório público no GitHub.

Você Pode Acessar o meu perfil no Linkedin.

Mas para que os site não sobreponha a página atual, é importante passar os parâmetros, target="_blank" e rel="external", para abrir em outra aba do navegador. Exemplo:

Você pode Acessar meu repositório público no GitHub.

Você Pode Acessar o meu perfil no Linkedin.

Usando Links Internos

Está é a primeira página do site. Se você quiser, pode acessar também a segunda página do site, aqui: Segunda Página

Para linkagem interna, não tem o rel="internal", porém, tem o rel="next" para ir para a próxima página e rel="prev" para voltar a página anterior. E isso ajuda os "Robos dos buscadores" a lerem o site melhor também.

Outra Observação é que, caso você não confie em algum link externo que você está direcionando, usa-se como boa prática o parâmetro rel="nofollow". Isso é especialmente para os mecanismos de busca também. Exemplo:

Links Para Download

Para fazer download de um arquivo, utiliza os argumentos download="nome do arquivo" target="application/pdf" ou zip. Aqui é importante destacar que os tipos de mídia não são todos os mesmos, então para ver todos os tipos, acessar o site iana.org E ir na página, Media Types
