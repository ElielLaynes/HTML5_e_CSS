Imagens Dinâmicas

Para fazer as imagens se adaptaram dinmicamente a tela, são usadas 2 tags. A PICTURE, tem o intuito de receber várias imagens dentro dela e a tag SOURCE MEDIA, serve configura as imagens.

IMPORTANTE: As imagens tem que estão na ordem, debaixo pra cima, da maior para a menor, se não vai quebrar a visualização.

O scirpt pra ela manipulação de imagem em HTML, fica da seguinte forma: 

        
    source media="(max-width: 750px)" srcset="foto1-p.png" type="image/png">
    source media="(max-width: 1050px)" srcset="foto2-m.png" type="image/png">
    img src="foto3-g.png" alt="Imagem flexivel">

Tente abrir esse site em vários dispositivos diferentes ou simplesmente amento e dinimuna o tamanho do seu navegador.
