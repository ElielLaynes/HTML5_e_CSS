Exemplo de CSS Inline

No exemplo abaixo, temos a técnica inline (a menos recomendada e se possível sempre evitar ela). A CSS inline, tem a função de fazer alterações de estilo dentro de uma tag, seja de paragramos, título e etc. Porém, o CSS inline, deixa o códgio poluído e feio, além de não ser nada escalável. visto que te que aplicar mudanças tag a tag
