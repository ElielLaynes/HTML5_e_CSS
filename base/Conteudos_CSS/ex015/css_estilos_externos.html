Exemplo de CSS Externo

A CSS externa é a mais indicada pra se usar. É criada atraves da tag link:css na head e referência um arquivo de css style separado, onde todas as mudanças são feitas no arquivo style e são aplicas a todas as páginas ao mesmo tempo.

Pode-se usar somente o arquvio style.css para as modificações, ou também pode-se se utilizada em conjunto com a CSS local ou interna :D
