Exemplo de CSS Local ou Interno

A CSS local ou interna é recomentado para projetos menores sem tanta complexidade. Difine-se a tag style na head e depois disso é só criar seletores dentro da tag style para as partes do códigos disponíveis no body, ou até mesmo outras várias tags disponiveis no body do HTML. A mudança feita em um seletor vai impactar todas as tags daquele seletor
