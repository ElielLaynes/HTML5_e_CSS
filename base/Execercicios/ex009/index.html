Código Fonte e Pré-Formatação

O camando document.getElementById('teste') é escrito em linguagem JavaScript.

Exemplo usando diretamente a tag CODE com um script Python.

num = int(input('Digite um número: ')) if num % 2 == 0: print(f'O número {num} é PAR.) else: print(f'o número {num} é ÍMPAR') print('Fim do Programa')

O "Problema" de usar direto a tag CODE é que ela não organiza o código e deixa ele escrito em linha, o que dificulta a leitura do código, como no exmplo acima.

Para solucionar esse ponto, pode-se usar a tag PRE, exemplo: Pega a Tag CODE e envelopa ela dentro da tag PRE e o código vai ficar organizado para leitura como no exemplo abaixo.

        
            num = int(input('Digite um número: '))
            if num % 2 == 0:
                print(f'O número {num} é PAR.)
            else:
                print(f'o número {num} é ÍMPAR')
            print('Fim do Programa')

Para deixar o código mais perdo da borda esquerda, é só tirar os espaços de tabulação. Caso queria acrescentar espaço, seleciona todo o código novamente e da um TAB

        
    num = int(input('Digite um número: '))
    if num % 2 == 0:
        print(f'O número {num} é PAR.)
    else:
        print(f'o número {num} é ÍMPAR')
    print('Fim do Programa')
