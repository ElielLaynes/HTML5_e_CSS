Trabalhando com Listas em HTML

Listas Ordenadas

Com a tag OL (order list), cria-se uma Lista Ordenada. Para cada Item da lista, usa-se a tag LI (list item)

  1. Acordar;
  2. Tomar Café;
  3. Limpar a casa;
  4. Colocar a Roupa para lavar;
  5. Fazer Almoço.

Dessa forma, não precisa ficar trocando os números caso seja precisa acrescentar outro intem na lista fora de ordem.

Listas Não Ordenadas

Para listas Não Ordenadas, usar-se a tag UL (Unordered List). Em relação ao type=" " dentro da tag UL, só há 3 opções: disc (Círculo preenchido), circle (Círculo sem preenchimento), square (Quadrado preenchido). O Padrão é o disc.

  • Pão
  • Leite
  • Tomate
  • Arroz
  • Feijão

lista de Definição

Funciona com um dicionário. Coloca-se o Termo e logo depois a Definição dele. Para criar uma lista de definição usa a tag DL (Definition list). Depois de criar a DL, coloca duas tags dentro, a DT (Definition Term) e DD (Definition Description)
